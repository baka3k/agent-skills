# Wiki Page Template

Standard structure for all wiki pages generated by wiki-generator.

## Page Structure

```markdown
# Page Title — {ProjectName}

> Evidence sources: `{mcp_refs}`, `{filesystem_refs}`

## Section 1: Overview
Brief summary of the page content (2-3 sentences max).

## Section 2-N: Content
Structured content with:
- Tables for structured data
- Code blocks for commands and examples
- Mermaid diagrams for visual relationships
- Cross-reference links to other pages

## Cross-References
- [Related Page 1](./related-1.md)
- [Related Page 2](./related-2.md)
- Back to [Index](./index.md)

---

_Generated by wiki-generator — Evidence confidence: {high|medium|low}_
```

## Writing Guidelines

1. **Every claim must have evidence** — tag source: mind_mcp, graph_mcp, filesystem
2. **Tables over paragraphs** — structured data is more scannable
3. **Code blocks for commands** — all setup/API examples in code blocks
4. **Mermaid for diagrams** — wiki-compatible, no external dependencies
5. **Cross-links mandatory** — no orphan pages, always link back to index
6. **One page, one purpose** — don't mix architecture with setup on same page
