# Workflow Contract

Pipeline:

1. Intake contracts and validate schema.
2. Retrieve evidence from MCP + trusted web.
3. Build two solution options.
4. Run effort and cost estimation.
5. Build staffing plan per delivery phase.
6. Run quality gates.
7. Generate proposal, estimate JSON, staffing plan, quality report, evidence log, and slide deck.

Confidence scoring policy:

- High: strong coverage from mind + graph + internet.
- Medium: two major evidence channels available.
- Low: one channel only, or major unknowns without supporting evidence.
