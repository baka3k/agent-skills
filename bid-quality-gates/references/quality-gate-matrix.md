# Quality Gate Matrix

| Gate | Pass Criteria |
| --- | --- |
| Scope clarity | Scope in/out and business goals are explicit and non-conflicting. |
| Architecture readiness | Key constraints, integration assumptions, and NFR posture are documented. |
| Estimation confidence | Best/base/worst PM exists with evidence-backed drivers. |
| Delivery readiness | Staffing and milestone sequencing are feasible against target timeline. |
| Production readiness | Compliance and operations controls are defined for go-live. |
